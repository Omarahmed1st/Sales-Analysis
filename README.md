# Big Data Sales Analysis (Superstore Dataset)

## Overview
This project analyzes a sales dataset to generate business insights such as total revenue, top-selling products, top cities by sales, profit by customer segment, and monthly sales trends.

## Key Insights
- Total Revenue: **$2.29M**
- Top City by Sales: **New York City**
- Most Profitable Segment: **Consumer**
- Top Product by Sales: **Canon imageCLASS 2200 Advanced Copier**
- Monthly sales trend shows an overall upward pattern over time.

## Tools & Libraries
- Python
- Pandas
- Matplotlib

## Project Structure
- `sales_analysis.py`: main analysis script
- `outputs/`: generated charts and screenshots
- `requirements.txt`: project dependencies
